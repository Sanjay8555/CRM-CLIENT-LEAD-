🚀 CRM System
📌 Overview

A simple Customer Relationship Management (CRM) system to manage complaints, track issues, and improve communication between users and administrators.

🎯 Features

Unique complaint ID for tracking

Real-time issue reporting

Automated complaint routing

Admin dashboard

Status updates & notifications

🛠️ Tech Stack

HTML, CSS, JavaScript, PHP/Java, MySQL

⚙️ Usage

Submit a complaint

Get a tracking ID

Admin processes the request

Track status until resolved

This contains everything you need to run your app locally.


## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
